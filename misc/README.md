## Update Release Checklist
1. Compile and move `/misc/changelog.md` into `/changelog.md` and wiki spreadsheet
2. Write update summary headline and edit wiki version tables (about page versions, history page)
3. Zip pack and worldgen, rename version, place in world and zip, archive zips with releases
4. Write formal organized changelog (see releases page)
5. Post updates to Modrinth pack, CurseForge pack, CurseForge World
6. Update modpack files with newest version details
7. Post updates to Modrinth modpack, CurseForge modpack
8. Update PlanetMC with update details/changes
9. Create GitHub tag and publish release
10. Announce update release publicly
